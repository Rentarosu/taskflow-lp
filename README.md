# TaskFlow - Landing Page

架空のプロジェクト管理SaaS「TaskFlow」のランディングページです。

> **注意:** これは実在するサービスではありません。ポートフォリオ掲載用のデモサイトです。

## 概要

チームのプロジェクト管理を効率化するSaaSツールを想定し、サービスサイトとして必要な全15ページを制作しました。

## ページ構成

| ページ | ファイル | 内容 |
|---|---|---|
| トップ | `index.html` | ヒーロー、機能紹介、導入事例、CTA |
| 機能紹介 | `features.html` | 主要機能の詳細 |
| 料金プラン | `pricing.html` | プラン比較・FAQ |
| 導入事例 | `case-studies.html` | 企業の導入ストーリー |
| 連携サービス | `integrations.html` | 外部ツールとの連携一覧 |
| ブログ | `blog.html` | 記事一覧 |
| アップデート | `updates.html` | リリースノート |
| APIドキュメント | `api-docs.html` | API仕様 |
| ヘルプセンター | `help.html` | FAQ・サポート |
| 会社概要 | `about.html` | チーム・ミッション |
| 採用情報 | `careers.html` | 求人一覧 |
| お問い合わせ | `contact.html` | 問い合わせフォーム |
| ログイン | `login.html` | ログイン画面 |
| プライバシーポリシー | `privacy.html` | 個人情報保護方針 |
| 利用規約 | `terms.html` | サービス利用規約 |

## 技術構成

- **HTML / CSS / JavaScript** のみで構成（フレームワーク不使用）
- CSS カスタムプロパティによるデザイントークン管理
- レスポンシブ対応
- スクロール連動のフェードインアニメーション
- Google Fonts（Inter / Noto Sans JP）

## ローカルでの確認方法

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/demoLP.git

# 任意のHTMLファイルをブラウザで開く
open index.html
```
