新機能 改善

AIタスクアシスタント & ダッシュボード刷新

AIを活用したタスク提案機能を新搭載。過去のプロジェクトデータを学習し、タスクの分割・見積もり・担当者割り当てを自動提案します。分析ダッシュボードも大幅にリニューアルし、より直感的なデータ可視化を実現しました。

  • AIベースのタスク自動分割・見積もり提案機能
  • 分析ダッシュボードのUI刷新（新チャートタイプ追加）
  • タスクテンプレートのカテゴリ分類とおすすめ表示
  • Slackインテグレーションのスレッド対応
  • ダークモードのコントラスト比改善
2026年2月5日
v3.7.0
新機能 バグ修正

タイムトラッキング機能の強化

タイムトラッキング機能を大幅にアップデート。週次・月次の自動レポート生成、プロジェクト横断でのリソース配分分析、見積もりと実績の乖離アラートなど、チームの稼働管理に欠かせない機能を追加しました。

  • タイムトラッキングの週次・月次レポート自動生成
  • プロジェクト横断リソース配分ヒートマップ
  • 見積もり時間との乖離検出アラート機能
  • ガントチャートのドラッグ操作が特定条件下で動作しない不具合を修正
  • モバイルアプリのプッシュ通知が重複する問題を修正
2026年1月15日
v3.6.0
新機能 改善

ワークフロー自動化エンジン刷新

ワークフロー自動化機能を根本からリアーキテクチャ。条件分岐の複雑なルール設定や、複数アクションの並列実行が可能になりました。テンプレートライブラリも拡充し、よくあるワークフローをワンクリックで導入できます。

  • 自動化ルールの条件分岐（IF/ELSE）対応
  • 複数アクションの並列実行サポート
  • 自動化テンプレートライブラリ（20種類追加）
  • Webhook送信のリトライ機能とログ表示
  • ダッシュボードの読み込み速度を40%改善
2025年12月8日
v3.5.0
改善 バグ修正

モバイルアプリ大型アップデート

iOS / Androidアプリを全面リニューアル。オフラインモードの強化、生体認証対応、ウィジェット機能の追加など、モバイル体験を大幅に向上させました。外出先でもデスクトップと同等の操作性を実現します。

  • モバイルアプリのUIを完全リデザイン
  • オフラインモードでのタスク作成・編集に対応
  • Face ID / Touch ID によるセキュアログイン
  • ホーム画面ウィジェット（本日のタスク一覧）
  • 画像アップロード時にアプリがクラッシュする問題を修正
  • 通知設定が正しく保存されない不具合を修正
2025年11月12日
v3.4.0
新機能 改善 バグ修正

Enterprise向けセキュリティ機能強化

大規模組織向けのセキュリティ機能を大幅に拡充。SAML 2.0ベースのシングルサインオン、きめ細かなカスタムロール設定、詳細な監査ログ機能を追加しました。SOC 2 Type II認証も取得完了し、エンタープライズ環境での安心運用をサポートします。

  • SAML 2.0 / SSO対応（Okta, Azure AD, Google Workspace）
  • カスタムロール・きめ細かな権限管理
  • 詳細な監査ログ（90日間保持、CSV/JSONエクスポート）
  • IP制限とデータリージョン選択機能
  • SOC 2 Type II認証取得
  • 大量タスク一括更新時のパフォーマンスを改善
  • フィルター条件の保存が一部正常に動作しない問題を修正
